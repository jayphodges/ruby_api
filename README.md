### Ruby Goldberg Machine API
 A docker deployable Rails API serving the most common names from the census from 2000.

### Prerequisites to Run Locally

  * [Docker](www.docker.com)

### Running Locally

  * Clone this repo.
  * ```docker-compose build```
  * ```docker-compose up```
  * visit ```localhost:8080```
